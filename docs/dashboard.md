1. `core`를 서비스 레이어로 추출(입출력 독립화)

3. disconnect 시 프로세스 정리 (고아 프로세스 방지)

5. Plan 모드 비활성화
6. CLI 의존성 명시 (서버 머신에 codex/claude 설치 필수)

결론: "가능하다"는 명확하고, 경로도 수렴했습니다. 단, MVP가 진짜 동작하는 MVP가 되려면 취소 경로 설계가 선택이 아니라 필수 범위입니다.
