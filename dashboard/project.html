Idea Studio
기획 캔버스

신규 기획이나 막연한 아이디어를 구조화해, 질문 없이도 제안·정제·검증 토론을 시작할 수 있습니다.

자동 질문 미리보기

기획 캔버스를 채우면 자동 생성되는 토론 질문이 여기에 표시됩니다.

입력한 내용은 세션에 idea-studio.txt 첨부로 함께 전달되어, 아이디어 제안·정리·검증 질문 도출에 활용됩니다.

Discussion은 합의점, 남은 쟁점, 다음 액션 중심으로 정리합니다.
현재 로컬 작업 디렉터리를 기본값으로 채웁니다. 순수 기획만 다룰 때는 고급 설정에서 프로젝트 자동 컨텍스트를 끌 수 있습니다.
PRD 초안, 기획 메모, 화면 스케치, 데이터 샘플을 함께 넣을 수 있습니다.

Template은 기존 역할 묶음을 유지하고, Custom은 2~6명의 역할과 모델을 직접 배치합니다.

역할 템플릿을 불러오는 중입니다.

같은 역할을 다른 모델에 중복 배치하거나, 같은 모델을 여러 슬롯에 반복 배치할 수 있습니다.

역할과 모델을 선택하면 구성이 여기에 요약됩니다.
Template preset은 유지되고, Custom 구성에서는 2~6명까지 직접 편성할 수 있습니다.
